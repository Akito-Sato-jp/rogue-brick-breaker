# Roguelike Brick Breaker
Pythonとtkinterで制作した、ローグライク要素を取り入れた進化型ブロック崩しです。

## 🎥 動作デモ
![ブロック崩しのデモ](./Brick.gif)
※プレイごとに変化するパワーアップを選択し、戦略的にステージを攻略します。

## 🧠 プロジェクトの特徴：戦略性とゲームデザイン
従来のブロック崩しに「ローグライク」のランダム性と選択の重要性を加え、リプレイ性の高いゲームを目指しました。

### 実装した主な機能
- **ローグライクな強化システム**: ステージクリアごとにランダムな3つの選択肢が出現。パドルの拡大、貫通弾、マルチボールなど、その時の状況に応じたビルド構築が楽しめます。
- **オブジェクト指向設計**: クラス継承（Inheritance）を活用し、特殊な挙動を持つブロックやアイテムを効率的に管理・実装。
- **衝突判定アルゴリズム**: tkinterのキャンバス座標を利用した、正確なバウンドとヒット判定のロジック。

## 🛠 使用技術
- **Language:** Python 3.11
- **Library:** tkinter
- **Key Concepts:** オブジェクト指向プログラミング (OOP), アルゴリズム設計
