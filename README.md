# Rogue-lite Brick Breaker
Pythonとtkinterで制作した、ローグライト要素を取り入れた進化型ブロック崩しです。

## 🎥 動作デモ
![ブロック崩しのデモ](./Brick.gif)
※パワーアップを選択しながらステージを攻略していくゲームプレイが特徴です。

## 🧠 プロジェクトの特徴：拡張性とゲームデザイン
従来のブロック崩しに「選択による強化」を加え、単調なプレイにならないよう工夫しました。

### 実装した主な機能
- **パワーアップ選択システム**: ステージクリアごとに、パドルの拡大、ボールの加速、弾丸の発射などの能力をランダムに選択可能。
- **オブジェクト指向設計**: クラス継承（Inheritance）を活用し、異なる種類のブロックやアイテムを効率的に管理・実装。
- **衝突判定アルゴリズム**: tkinterのキャンバス座標を利用した、正確なバウンドとヒット判定のロジック。

## 🛠 使用技術
- **Language:** Python 3.11
- **Library:** tkinter
- **Key Concepts:** オブジェクト指向プログラミング (OOP), ゲームループの設計

## 📂 ファイル構成
- `main.py`: ゲームのメインプログラム
- `Brick.gif`: 動作デモ画像
- `assets/`: 効果音や画像素材（もしあれば）

## 🎮 実行方法
1. 本リポジトリをクローンします。
2. 以下のコマンドで実行してください。
   `python main.py`
